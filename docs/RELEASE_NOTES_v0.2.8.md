# Release Notes v0.2.8

`v0.2.8`은 starter kit 폴더명을 교사 초보자 기준으로 더 자연스럽게 정리한 BRAT 배포 태그입니다.

## 포함 변경

- starter kit의 `Templates/homepage`, `Samples/homepage` 폴더명을 각각 `템플릿/homepage`, `예시/homepage`로 변경
- 빠른 시작 노트, 샘플 노트, 사용자 문서의 폴더 경로를 한글 이름 기준으로 정리
- starter kit 재설치 시 관리 대상 원본 노트를 최신 starter kit 구조로 갱신하도록 보완

## BRAT 설치 범위

- 이 태그는 `homepage` 플러그인 설치용입니다.
- starter kit은 플러그인 설치 후 Obsidian 안에서 `초보자용 시작 세트 설치` 명령으로 vault에 생성됩니다.

## 설치 확인

설치 후 starter kit을 실행하면 아래 경로가 보여야 합니다.

- `템플릿/homepage/학생 메모 템플릿.md`
- `템플릿/homepage/상담 기록 템플릿.md`
- `예시/homepage/학생 메모 예시.md`
- `예시/homepage/상담 기록 예시.md`
