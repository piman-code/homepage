# Release Notes v0.2.6

`v0.2.6`은 `homepage` 플러그인의 현재 운영 구조를 정리한 BRAT 배포 태그입니다.

## 포함 변경

- 플러그인 id를 `homepage` 기준으로 정리
- 학급 홈페이지 대시보드를 `오늘의 공지 / 오늘의 출석 / 우리반 상점 / 우리반 리포트` 4개 카드 구조로 정리
- 카드 아래 `오늘 한 줄 요약` 자동 동기화 반영
- 학생 관계 그래프를 교사용 노트로 분리
- 문서, 문제 해결 가이드, 스펙 문서의 현재 구조 반영

## BRAT 설치 범위

- 이 태그는 `homepage` 플러그인 설치용입니다.
- clean live vault, Google Form 자동 수집 watcher, 학생 포털 배포본은 별도 운영 번들입니다.

## 설치 확인

설치 후 커맨드 팔레트에서 아래 명령이 보여야 합니다.

- `학급 홈페이지 열기`
- `오늘 공지 섹션 추가`
- `학급 기본 구조 재생성(백업 후 덮어쓰기)`
- `폼 링크 자동 적용`
- `주간 자동 보고서 생성`
