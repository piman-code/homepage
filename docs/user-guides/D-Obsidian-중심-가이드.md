# D. Obsidian 중심 가이드

이 문서는 `homepage`를 Obsidian 안에서 가장 자연스럽게 쓰는 방법을 설명합니다.

## 1단계. 추천 폴더 구조

starter kit을 설치하면 기본적으로 아래 흐름을 쓰면 됩니다.

```text
홈/
1. 공지사항/
2. 주간학습안내/
3. 뉴스읽기/
4. 수업활동/
5. 설문/
6. 학생성장/
7. 상담기록/
8. 과제관리/
9. 수업자료/
10. 활동기록/
Templates/homepage/
Samples/homepage/
00. 시작하기/
```

## 2단계. 어떤 파일을 어디에 두면 좋은가

- 오늘 학부모/학생 공지: `1. 공지사항/`
- 주간 계획/링크 모음: `2. 주간학습안내/`
- 기사 과제/시사 활동: `3. 뉴스읽기/`
- 학생 성장 요약/칭찬 후보: `6. 학생성장/`
- 상담 노트: `7. 상담기록/`
- 과제 점검: `8. 과제관리/`
- 수업 준비: `9. 수업자료/`
- 활동 직후 기록: `10. 활동기록/`
- 복붙 원본 템플릿: `Templates/homepage/`

## 3단계. 템플릿 복붙 방법

플러그인 없이 하는 가장 쉬운 방법:

1. `Templates/homepage/학생 메모 템플릿.md`를 엽니다.
2. 전체 복사합니다.
3. 새 노트를 만듭니다.
4. 붙여 넣습니다.
5. 파일명을 날짜 포함 형식으로 바꿉니다.

예시 파일명:

- `7. 상담기록/2026-03-09-학생메모-김예시.md`
- `8. 과제관리/2026-03-09-과학과제-점검.md`
- `9. 수업자료/2026-03-09-사회-토의준비.md`

## 4단계. 플러그인 없이도 되는 방법

기본 흐름:

1. `homepage` 플러그인 설치
2. 기본 구조 생성
3. starter kit 설치
4. 템플릿 복붙
5. 위키링크로 노트 연결

이 상태만으로도 충분히 운영 가능합니다.

## 5단계. 플러그인이 있으면 더 좋아지는 방법

### Dataview

상담 기록 목록을 자동으로 모으고 싶을 때 좋습니다.

예시:

```dataview
TABLE file.mtime as 수정일
FROM "7. 상담기록"
SORT file.mtime desc
```

### Templater

새 노트를 만들 때 날짜를 자동으로 넣고 싶을 때 좋습니다.

예시:

```md
날짜: <% tp.date.now("YYYY-MM-DD") %>
학생:
상황:
```

### QuickAdd

`학생 메모 만들기`, `상담 기록 만들기` 같은 빠른 캡처 버튼을 만들 때 좋습니다.

추천 QuickAdd 항목:

- 학생 메모 생성
- 상담 기록 생성
- 과제 점검 노트 생성

## 6단계. Dataview / Templater / QuickAdd 연결 가능성

가장 추천하는 연결 순서:

1. 먼저 `homepage`만으로 운영
2. 목록 정리가 필요하면 Dataview 추가
3. 템플릿 자동 입력이 필요하면 Templater 추가
4. 버튼 캡처가 필요하면 QuickAdd 추가

중요:

- 이 세 플러그인은 `필수`가 아닙니다.
- 없어도 `homepage`의 핵심 흐름은 작동해야 합니다.

## 7단계. Obsidian 초보자가 자주 헷갈리는 점

- `경로`는 Finder 전체 경로가 아니라 vault 안쪽 경로입니다.
- 다만 `homepage` 설정은 절대경로를 붙여 넣어도 현재 vault 기준으로 자동 정리하도록 개선하는 것이 좋습니다.
- 홈페이지는 “메인 화면”, 실제 내용은 각 개별 노트에 적는 구조입니다.

## 8단계. Obsidian 중심으로 쓰는 추천 루틴

아침:

1. `홈/홈페이지.md` 열기
2. 오늘 공지 열기
3. 수업 준비 노트 열기

낮:

1. 학생 메모 1건 작성
2. 활동 기록 1건 작성

퇴근 전:

1. 주간 자동 보고서 생성
2. 다음 날 확인할 노트만 남기기
