# C. 수업용 사용 가이드

이 문서는 `수업 자료 정리`, `활동 기록`, `학생별/모둠별 메모`, `복습/피드백 기록`을 `homepage` 중심으로 운영하는 방법을 설명합니다.

## 1단계. 수업용으로 자주 쓰는 폴더

- `9. 수업자료/`: 차시별 수업 준비 노트
- `10. 활동기록/`: 활동 후 짧은 기록
- `3. 뉴스읽기/`: 기사 기반 읽기/토론 활동
- `예시/homepage/`: 예시 완성본 보기
- `템플릿/homepage/수업 준비 템플릿.md`: 가장 자주 쓸 틀

## 2단계. 수업 자료 정리 방법

1. `템플릿/homepage/수업 준비 템플릿.md`를 엽니다.
2. 새 노트를 `9. 수업자료/2026-03-09-사회-모둠토의.md`처럼 만듭니다.
3. 아래 4가지만 먼저 씁니다.

- 날짜
- 교과
- 오늘 목표
- 준비물

간단 예시:

```md
# 2026-03-09 사회 모둠토의 준비

- 오늘 목표: 서로 다른 관점을 비교해 본다.
- 준비물: 활동지, 색펜, 타이머
- 도입: 오늘 기사 1개 읽기
- 활동: 모둠 토의 15분
- 정리: 모둠별 한 줄 발표
```

## 3단계. 활동 기록 남기기

수업 직후 3분 기록법:

1. `10. 활동기록/2026-03-09-사회-토의-기록.md` 같은 노트를 만듭니다.
2. 아래 세 줄만 적습니다.

- 잘 된 점
- 어려웠던 점
- 다음 시간 보완점

예시:

```md
- 잘 된 점: 1모둠이 역할 분담을 빨리 했다.
- 어려웠던 점: 4모둠은 시작이 늦었다.
- 다음 시간 보완점: 발언이 적은 학생에게 먼저 짧은 역할을 준다.
```

## 4단계. 학생별 메모 남기기

1. `학생 메모 템플릿`을 복사합니다.
2. 특정 학생이 보인 반응이나 질문을 짧게 적습니다.
3. 공개 노트에는 실명 민감정보를 넣지 않습니다.

추천 기록 포인트:

- 질문을 많이 한 학생
- 발표를 회피한 학생
- 모둠 조정 역할을 잘한 학생
- 개별 지원이 필요한 학생

## 5단계. 모둠별 메모 남기기

모둠별로 적을 때는 아래 형식이 편합니다.

```md
# 2026-03-09 사회 3모둠 메모

- 참여도: 보통
- 발표 준비: 좋음
- 자료 정리: 보통
- 다음 시간 역할 제안: 발표 1명, 기록 1명, 질문 1명
```

## 6단계. 복습/피드백 기록 예시

하교 전 2분 피드백:

1. 오늘 수업 자료 노트를 엽니다.
2. 맨 아래에 `복습 메모` 섹션을 추가합니다.
3. 다음 수업 때 바꿀 점만 한두 줄 적습니다.

예시:

```md
## 복습 메모
- 예시 답안을 1개 더 보여 주면 시작 속도가 빨라질 것 같다.
- 질문 활동이 좋았으므로 다음 시간에도 질문 카드를 유지한다.
```

## 7단계. 뉴스읽기 수업 예시

1. `오늘자 뉴스읽기 과제 생성`을 실행합니다.
2. 기사 제목/출처/링크를 넣습니다.
3. 수업 전에는 `핵심 요약` 칸만 먼저 채웁니다.
4. 수업 후에는 학생 토론 질문 중 좋았던 것을 활동 기록에 남깁니다.

## 8단계. 수업 전-중-후 워크플로우

수업 전:

1. 홈페이지 열기
2. 오늘 공지 확인
3. 수업 준비 노트 열기

수업 중:

1. 학생 메모 1~2건만 기록
2. 모둠별 특징 1줄씩 기록

수업 후:

1. 활동 기록 3줄 작성
2. 다음 차시 준비물 수정

## 9단계. 초보자에게 추천하는 최소 기록 원칙

- 한 수업에 한 노트만 남겨도 충분합니다.
- 완벽한 기록보다 “다음 시간에 도움이 되는 기록”을 남기세요.
- 활동 기록은 3줄이면 충분합니다.
