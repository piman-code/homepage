## 가장 쉬운 시작 순서

1. Obsidian에서 `homepage` 플러그인을 설치합니다.
2. `Settings -> homepage -> 빠른 시작`으로 갑니다.
3. `1단계. 기본 구조 만들기`를 누릅니다.
4. `2단계. 초보자용 시작 세트 설치`를 누릅니다.
5. `3단계. 학급 홈페이지 열기`를 누릅니다.
