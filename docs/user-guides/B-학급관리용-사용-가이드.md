# B. 학급관리용 사용 가이드

이 문서는 담임교사가 `출석`, `학생 메모`, `상담 기록`, `과제 관리`, `주간 운영`을 Obsidian 안에서 바로 굴리는 방법을 설명합니다.

## 1단계. 추천 폴더 역할 이해하기

- `홈/홈페이지.md`: 오늘 들어가는 첫 화면
- `1. 공지사항/`: 학부모/학생 공지
- `2. 주간학습안내/`: 주간 보고, 주간 안내
- `3. 뉴스읽기/`: 시사 과제, 기사 기반 활동
- `6. 학생성장/`: 요약 데이터와 성장 신호
- `7. 상담기록/`: 학생 상담 기록
- `8. 과제관리/`: 제출 점검 메모
- `템플릿/homepage/`: 복붙용 틀

## 2단계. 출석 운영 예시

가장 쉬운 방법:

1. 아침에 `학급 홈페이지 열기`를 누릅니다.
2. 홈페이지의 `오늘의 출석` 구역을 확인합니다.
3. 아직 데이터가 없으면 `학생 성장 요약 불러오기`를 실행합니다.
4. 실제 출석 메모는 별도 노트에 짧게 남깁니다.

교사용 간단 메모 예시:

```md
# 2026-03-09 출석 메모

- 결석: 1명
- 지각: 2명
- 보건실 방문 예정: 1명
- 1교시 전 확인할 학생: 김예시, 박예시
```

## 3단계. 학생 메모 운영 예시

1. `템플릿/homepage/학생 메모 템플릿.md`를 엽니다.
2. 전체 내용을 복사합니다.
3. `7. 상담기록/2026-03-09-학생메모-김예시.md` 같은 새 노트를 만듭니다.
4. 붙여 넣고 날짜, 학생 이름, 상황만 채웁니다.

바로 쓸 문장 예시:

- 수업 시작은 느렸지만 발표 참여는 좋았다.
- 과제는 아직 제출하지 않았지만 이유를 분명히 설명했다.
- 쉬는 시간에 같은 모둠 친구를 잘 도왔다.

## 4단계. 상담 기록 운영 예시

1. `템플릿/homepage/상담 기록 템플릿.md`를 엽니다.
2. 새 노트를 `7. 상담기록/2026-03-09-상담-박예시.md`처럼 만듭니다.
3. `들은 내용`, `교사 판단`, `후속 조치`만 먼저 적습니다.

상담 기록 예시:

```md
## 들은 내용
- 수학 숙제를 어디서부터 시작해야 할지 모르겠다고 말했다.

## 교사 판단
- 바로 도와줄 것: 1번 문제 시작을 함께 해 준다.
- 추후 확인할 것: 금요일까지 혼자 시작할 수 있는지 본다.
```

## 5단계. 과제 관리 예시

1. `템플릿/homepage/과제 관리 템플릿.md`를 엽니다.
2. `8. 과제관리/2026-03-09-과학과제-점검.md` 같은 노트를 만듭니다.
3. 제출 완료/미제출/재안내 필요 학생만 적습니다.

과제 관리 예시:

```md
# 과학 탐구 과제 점검

- 제출 완료: 20명
- 미제출: 김예시, 이예시
- 재안내 필요: 박예시
- 메모: 사진 첨부 방법을 다시 안내하기
```

## 6단계. 교사가 바로 쓸 수 있는 템플릿 묶음

starter kit 설치 후 바로 쓰는 파일:

- `템플릿/homepage/학생 메모 템플릿.md`
- `템플릿/homepage/상담 기록 템플릿.md`
- `템플릿/homepage/수업 준비 템플릿.md`
- `템플릿/homepage/과제 관리 템플릿.md`

추천 규칙:

1. 템플릿 원본은 수정하지 않습니다.
2. 새 노트를 만든 뒤 복붙해서 씁니다.
3. 파일명 앞에 날짜를 넣습니다.

## 7단계. 주간 운영 예시

### 월요일

1. `주간 자동 보고서 생성`
2. 이번 주 공지 초안 작성
3. 준비물/행사 정리

### 화요일

1. 학생 메모 2건 작성
2. 상담 필요 학생 체크

### 수요일

1. 뉴스읽기 과제 배포
2. 활동 중 관찰 학생 메모

### 목요일

1. 과제 제출 확인
2. 보호자 연락 필요 학생 정리

### 금요일

1. 주간 자동 보고서 다시 생성
2. 다음 주 이어볼 메모만 남기기

## 8단계. 초보자에게 추천하는 최소 운영법

처음에는 아래 네 가지만 해도 충분합니다.

1. 홈페이지 열기
2. 오늘자 공지 만들기
3. 학생 메모 1건 남기기
4. 과제 점검 노트 1건 만들기

## 9단계. 자주 하는 실수

- 모든 기록을 한 노트에 몰아넣기
- 날짜 없는 파일명 쓰기
- 공개용 공지 노트와 내부 상담 기록을 섞기
- Google Form이 없으면 시작 못 한다고 생각하기
