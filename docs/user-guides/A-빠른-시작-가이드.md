# A. 빠른 시작 가이드

이 문서는 `homepage`를 처음 설치한 교사가 가장 짧은 경로로 시작하도록 돕는 문서입니다.

## 1단계. 설치 준비

준비할 것:

- Mac에 설치된 Obsidian
- 새 vault 또는 이미 쓰는 수업용 vault
- 인터넷 연결

중요:

- 일반 사용자는 Node.js나 npm이 필요하지 않습니다.
- 개발용 명령은 무시하고, Obsidian 안에서만 따라가면 됩니다.

## 2단계. Obsidian 설치

1. Obsidian을 실행합니다.
2. 시작 화면에서 `Create new vault`를 누릅니다.
3. vault 이름을 예를 들어 `2026-1 우리반`처럼 정합니다.
4. 저장 위치를 정합니다.
5. `Create`를 누릅니다.

추천:

- 처음 테스트는 iCloud/Google Drive보다 로컬 폴더로 시작하세요.
- 한글 이름, 공백이 있는 경로도 사용 가능하지만 첫 테스트는 단순한 경로가 더 쉽습니다.

## 3단계. Community plugins 켜기

1. 왼쪽 아래 톱니바퀴 `Settings`를 누릅니다.
2. 왼쪽 메뉴에서 `Community plugins`를 누릅니다.
3. `Turn on community plugins`를 누릅니다.
4. 경고창이 나오면 확인합니다.

## 4단계. BRAT 설치

1. `Community plugins` 화면에서 `Browse`를 누릅니다.
2. 검색창에 `BRAT`를 입력합니다.
3. BRAT를 설치하고 활성화합니다.
4. BRAT 설정에서 `Add Beta plugin`을 누릅니다.
5. 저장소 주소에 아래 값을 붙여 넣습니다.

```text
https://github.com/piman-code/homepage
```

6. 설치가 끝나면 플러그인 목록에서 `homepage`를 켭니다.

## 5단계. 첫 실행

1. `Settings -> homepage`로 이동합니다.
2. 맨 위 `빠른 시작` 구역을 찾습니다.
3. 아래 버튼을 순서대로 누릅니다.

   1. `기본 구조 만들기`
   2. `시작 세트 설치`
   3. `학급 홈페이지 열기`

4. `홈/홈페이지.md`가 열리면 성공입니다.

## 6단계. 초보자가 바로 해 볼 기본 사용

### 오늘 공지 만들기

1. 커맨드 팔레트를 엽니다.
2. `오늘자 공지 노트 생성`을 실행합니다.
3. 열린 노트에서 `핵심 안내`, `일정/준비물`만 먼저 적습니다.

### 오늘 뉴스읽기 과제 만들기

1. 커맨드 팔레트를 엽니다.
2. `오늘자 뉴스읽기 과제 생성`을 실행합니다.
3. 기사 제목, 출처, 링크, 3줄 요약만 먼저 적습니다.

### 주간 문서 만들기

1. 커맨드 팔레트를 엽니다.
2. `주간 자동 보고서 생성`을 실행합니다.
3. 이번 주 핵심 링크가 모인 노트를 확인합니다.

## 7단계. Obsidian에서 starter kit 열어 보기

`시작 세트 설치`를 누르면 아래 파일이 vault 안에 생깁니다.

- `00. 시작하기/homepage 빠른 시작.md`
- `템플릿/homepage/*.md`
- `예시/homepage/*.md`

가장 먼저 볼 파일:

1. `00. 시작하기/homepage 빠른 시작.md`
2. `예시/homepage/학생 메모 예시.md`
3. `예시/homepage/상담 기록 예시.md`

## 8단계. Google Form이 아직 없어도 괜찮습니다

처음에는 아래 순서만 해도 충분합니다.

1. 홈페이지 만들기
2. 공지 노트 만들기
3. 학생 메모/상담 기록 템플릿 복붙하기

Google Form URL은 나중에 넣어도 됩니다.

## 9단계. 기본 사용 예시

아침에 할 일:

1. `학급 홈페이지 열기`
2. `오늘자 공지 노트 생성`
3. 준비물/시간표/전달사항 입력

수업 중 할 일:

1. `템플릿/homepage/학생 메모 템플릿.md`를 복사합니다.
2. 오늘 눈에 띈 학생 1~2명만 짧게 기록합니다.

하교 전 할 일:

1. `주간 자동 보고서 생성`
2. 오늘 공지와 과제 링크가 잘 연결됐는지 확인합니다.

## 10단계. 막히면 먼저 확인할 것

1. `Settings -> homepage -> 빠른 시작` 버튼이 보이는지
2. `홈/홈페이지.md` 파일이 생겼는지
3. 커맨드 팔레트에 아래 명령이 보이는지

- `학급 홈페이지 열기`
- `초보자용 시작 세트 설치`
- `오늘자 공지 노트 생성`
- `오늘자 뉴스읽기 과제 생성`

더 자세한 문제 해결은 `F 문서`를 보면 됩니다.
