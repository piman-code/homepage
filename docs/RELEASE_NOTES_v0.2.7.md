# Release Notes v0.2.7

`v0.2.7`은 다른 macOS 사용자 계정과 Obsidian 초보자 기준의 첫 실행 흐름을 보강한 BRAT 배포 태그입니다.

## 포함 변경

- `초보자용 시작 세트 설치` 명령 추가
- `Settings -> homepage -> 빠른 시작` 3단계 버튼 추가
- starter kit 형태의 시작 문서, 템플릿, 샘플 노트 자동 생성
- `홈페이지 노트 경로`, `뉴스읽기 폴더`에 붙여 넣은 절대경로를 현재 vault 기준으로 자동 정리
- 다른 계정에서 가져온 설정을 현재 vault 기준으로 보정하는 로직 추가
- 신규 macOS 계정 기준 빠른 시작, 학급관리, 수업용, Obsidian 중심, Google Apps Script, 문제 해결 문서 추가

## BRAT 설치 범위

- 이 태그는 `homepage` 플러그인 설치용입니다.
- starter kit은 플러그인 설치 후 Obsidian 안에서 `초보자용 시작 세트 설치` 명령으로 vault에 생성됩니다.

## 설치 확인

설치 후 커맨드 팔레트에서 아래 명령이 보여야 합니다.

- `학급 홈페이지 열기`
- `초보자용 시작 세트 설치`
- `오늘자 공지 노트 생성`
- `오늘자 뉴스읽기 과제 생성`
- `폼 링크 자동 적용`
- `주간 자동 보고서 생성`
